# WiFi Password Viewer

A Python CLI tool to view saved WiFi passwords and signal strength.

## Features:
- Connected WiFi detection
- Saved WiFi password viewer
- Signal strength visualization
- Export to file
- Colored terminal UI

## Requirements:
- Python 3.x
- Windows OS (netsh support)

## Run:
python wifi-password-viewer.py
